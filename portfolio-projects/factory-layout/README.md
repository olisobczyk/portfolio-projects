# Production System Planning & Factory Layout Optimization

**Tech Stack:** visTABLE · Operations Design · Cost Analysis  
**Type:** Academic project (Mechanical Engineering)  
**Scope:** Conceptual factory design for buggy vehicle assembly

## The Challenge

Design an efficient production facility layout balancing material flows, machine utilization, operational costs, and workspace constraints for a buggy vehicle assembly operation.

## The Solution

Conducted comprehensive factory design analysis using visTABLE, mapping material flows, machine working times, and operational costs. Developed optimized production and auxiliary space layouts to maximize throughput while minimizing waste and idle time.

## Key Achievements

- Designed material flow network across production stages
- Analyzed machine utilization and cycle time optimization
- Evaluated production and auxiliary space requirements
- Compared multiple layout variants on cost and efficiency criteria
- Delivered actionable recommendations for facility setup
