# CRM System & Operational Database Automation

**Tech Stack:** Google Sheets · SQL · Google Apps Script · Python · Claude & ChatGPT  
**Duration:** 1 year (ongoing)  
**Role:** Solo architect and developer  
**Company:** Metalpasja (manufacturing service)

## The Challenge

Manufacturing service company lacked centralized data infrastructure. All financial transactions, operational logs, and customer interactions were recorded manually in notebooks, creating blind spots across the organization. Impossible to estimate customer costs, track monthly expenses, or evaluate profitability by project.

## The Solution

Designed and implemented a relational database architecture in Google Sheets supporting customer management, order tracking, work scheduling, and financial analysis. Engineered 7+ automated workflows using Google Apps Script to eliminate manual reporting and streamline core business processes.

## Key Achievements

- **Automated Processes:** Customer cost reports, monthly operational summaries, employee expense tracking, client invoicing, daily backups, dynamic work scheduling
- **Database Architecture:** Designed interconnected sheet structure eliminating data duplication while maintaining referential integrity across growing datasets
- **Optimization:** Diagnosed and resolved performance bottlenecks through iterative refinement — replaced inefficient script-based solutions with embedded formulas to maintain system responsiveness as data volume increased
- **AI-Assisted Problem Solving:** Used Claude and ChatGPT to debug complex formula logic and evaluate alternative implementation strategies

## Impact

| Metric | Value |
|--------|-------|
| Profitability Increase | 100% |
| Automated Workflows | 7+ |

Financial turnover and profit-to-cost ratio **doubled within 12 months** of implementation.
